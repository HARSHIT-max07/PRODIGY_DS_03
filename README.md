# Decision Tree Classifier - Bank Marketing

This project builds a Decision Tree Classifier to predict whether a customer will purchase a product or service using demographic and behavioral data.

## Steps
- Data Cleaning
- Feature Encoding
- Train Test Split
- Decision Tree Model
- Model Evaluation

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)
